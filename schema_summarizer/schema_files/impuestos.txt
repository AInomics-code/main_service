TABLA: impuestos
DESCRIPCIÓN: Tabla que contiene información de los impuestos. Usa esta tabla cuando quieras analizar tipos de impuestos o calcular montos fiscales.

REGLAS DE NEGOCIO:
- Cada impuesto tiene un código único
- Los impuestos se aplican a productos y transacciones

EJEMPLOS DE PREGUNTAS PARA ESTA TABLA:
- Cuantos tipos de impuestos existen?
- Cuales son los impuestos con mayor porcentaje?

COLUMNAS:
- codigo_de_impuesto (nvarchar): Código único del impuesto
- nombre_de_impuesto (nvarchar): Nombre del impuesto
- porcentaje_de_impuesto (float): Porcentaje del impuesto
