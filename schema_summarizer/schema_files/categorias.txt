TABLA: categorias
DESCRIPCIÓN: Tabla que contiene las categorías de productos. Usa esta tabla cuando quieras analizar productos por categoría o segmentación de inventario.

REGLAS DE NEGOCIO:
- Cada categoría tiene un código único
- Las categorías agrupan productos para análisis y reportes

EJEMPLOS DE PREGUNTAS PARA ESTA TABLA:
- Cuantas categorias de productos existen?
- Cuales son las categorias con mas productos?

COLUMNAS:
- codigo_de_categoria (nvarchar): Código único de la categoría
- nombre_de_categoria (nvarchar): Nombre de la categoría
