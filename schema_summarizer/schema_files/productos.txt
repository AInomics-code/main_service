TABLA: productos
DESCRIPCIÃ“N: Esta tabla te permite buscar productos para saber su ID unico y asi poder relacionarlo con otras tablas. Lo mas probable es que el usuario no use busque por el nombre exacto como esta en la tabla por lo cual se debe tomar el mas cercano.

EJEMPLOS DE PREGUNTAS PARA ESTA TABLA:
- Cual es la ID del producto Salsa de tomate?

COLUMNAS:
- producto_id (nvarchar): ID unico del producto
- nombre (nvarchar): Nombre del producto
- estado (nvarchar): estado de disponibilidad o activacion del Producto
- costo (float): costo del producto
- marca (nvarchar): marca del producto
